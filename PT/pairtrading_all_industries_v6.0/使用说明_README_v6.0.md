# A股全行业配对交易策略 v6.0 使用说明

## 📋 功能概述

本系统实现了基于卡尔曼滤波的全行业配对交易策略，主要功能包括：

1. **全行业自动扫描**：自动读取通达信行业板块文件，遍历所有行业
2. **ST自动剔除**：通过股票名称识别并剔除ST股票
3. **智能配对筛选**：每个行业SSD+协整检验筛选候选配对
4. **Top2优选**：每个行业只保留收益率前2的配对
5. **完整回测报告**：输出多个CSV文件和可视化图表

## 📁 文件清单

| 文件名 | 用途 |
|--------|------|
| `pairtrading_all_industries_v6.0.py` | **主程序**：全行业回测 |
| `analyze_results_v6.0.py` | 分析脚本：生成可视化图表 |
| `verify_data_v1.0.py` | 验证脚本：检查数据路径 |

## 🚀 快速开始

### 第一步：数据验证（推荐）

```bash
python verify_data_v1.0.py
```

**功能**：
- 解析行业板块文件，统计行业数量
- 识别ST股票数量
- 检查通达信数据路径
- 输出行业列表统计CSV

**预期输出**：
```
行业总数: XX
股票总数(含ST): XXX
ST股票数: XX (X.X%)
有效股票数: XXX
```

### 第二步：运行全行业回测

```bash
python pairtrading_all_industries_v6.0.py
```

**回测流程**：
1. 遍历每个行业板块
2. 剔除ST股票
3. 样本内筛选Top 5配对（SSD+协整检验）
4. 样本外回测所有候选配对
5. 按收益率排序，保留Top 2
6. 输出结果到CSV

**输出文件**：
- `全行业配对回测详细结果_YYYYMMDD_HHMMSS.csv` - 完整数据
- `全行业配对分析表_YYYYMMDD_HHMMSS.csv` - 简洁分析表
- `全行业统计_YYYYMMDD_HHMMSS.csv` - 行业汇总统计

### 第三步：可视化分析

```bash
python analyze_results_v6.0.py
```

**生成图表**：
- `全行业收益率排名.png` - 各行业平均收益率横向条形图
- `收益风险分布分析.png` - 4合1分析图（收益分布、夏普分布、散点图、胜率分布）
- `Top20_配对排名表.png` - 带颜色标记的排名表格

## ⚙️ 配置参数

在主程序 `pairtrading_all_industries_v6.0.py` 中修改以下配置：

```python
# 数据路径
LOCAL_INDUSTRY_TXT_PATH = "C:/new_tdx/T0002/export/行业板块.txt"
TDX_DATA_DIR = "C:/new_tdx/vipdoc"

# 回测时间区间
IN_SAMPLE_START = "2018-01-01"   # 样本内开始
IN_SAMPLE_END = "2023-12-31"     # 样本内结束（训练期）
OUT_SAMPLE_START = "2024-01-01"  # 样本外开始（测试期）
OUT_SAMPLE_END = "2026-04-08"    # 样本外结束

# 筛选参数
TOP_N_PAIRS_PER_INDUSTRY = 5  # 每个行业候选配对数
FINAL_TOP_N = 2               # 每个行业最终保留数

# 策略参数
PARAMS = {
    'initial_capital': 1000000,    # 初始资金
    'commission_rate': 0.0003,      # 佣金率
    'stamp_tax_rate': 0.001,        # 印花税
    'entry_threshold': 1.2,         # 入场Z-Score阈值
    'exit_threshold': 0.3,        # 出场Z-Score阈值
    'stop_loss': 2.5,             # 止损阈值
    'max_holding_days': 15,       # 最大持仓天数
    'rebalance_threshold': 0.6,   # 轮动阈值
    'max_position_ratio': 0.6,    # 最大仓位比例
    'coint_check_freq': 30,       # 协整检验频率（天）
    'cooling_period': 2,          # 基础冷却期
    'max_cooldown': 10,           # 最大冷却期
    'min_coint_period': 60        # 最小协整检验周期
}
```

## 🔍 结果解读

### 优质配对特征
- **收益率 > 10%**：显著正收益
- **夏普比率 > 1**：风险收益比良好
- **最大回撤 < 10%**：风险控制较好
- **胜率 > 50%**：交易逻辑有效
- **协整P值 < 0.05**：统计上显著配对

### 行业筛选建议
- 优先选择**平均收益率>5%**的行业
- 关注**夏普比率>0.5**的行业
- 剔除**最大回撤>20%**的高风险行业

## ⚠️ 注意事项

1. **数据准备**：确保通达信日线数据完整下载
2. **内存要求**：全行业回测需要较大内存（建议16GB+）
3. **运行时间**：根据行业数量，可能需要30分钟-2小时
4. **路径检查**：如果数据路径不同，请修改配置中的路径
5. **ST识别**：通过股票名称识别ST，确保行业板块文件包含股票名称列

## 🐛 常见问题

### Q: 找不到数据文件？
A: 检查 `LOCAL_INDUSTRY_TXT_PATH` 和 `TDX_DATA_DIR` 路径是否正确

### Q: 所有行业都显示"有效股票不足"？
A: 检查通达信日线数据是否已下载，可尝试读取单只股票数据测试

### Q: 回测速度很慢？
A: 这是正常的，全行业回测涉及大量计算。建议先测试单个行业

### Q: 如何只回测特定几个行业？
A: 修改代码中的 `industry_map.items()` 循环，添加行业过滤条件

## 📈 策略逻辑

### 样本内筛选（训练期）
1. 计算所有配对的SSD（平方距离）
2. 按SSD排序取Top 15对
3. OLS回归估计对冲比率
4. ADF检验协整性（p<0.05）
5. 保留通过检验的配对

### 样本外交易（测试期）
1. 卡尔曼滤波动态更新对冲比率
2. Z-Score > 1.2：买入低估方
3. Z-Score < -1.2：买入另一只
4. |Z-Score| < 0.3：止盈出场
5. Z-Score反向突破：轮动换股
6. 定期协整检验，破裂则冷却
